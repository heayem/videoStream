API Integration & Features

Welcome to the Video Streamer Pro API documentation. This document covers the backend integration endpoints and the key features of the application.

Endpoints

  • GET /api/videos - Retrieve all videos in the library. Returns JSON array.
  • GET /api/videos/:id - Get details about a specific video.
  • POST /api/videos - Upload a new video. Accepts multipart/form-data with video, thumbnail, title, library, and collection fields.
  • PATCH /api/videos/:id - Update video fields (such as title).
  • DELETE /api/videos/:id - Remove a video and delete its underlying assets.
  • GET /stream/:id/master.m3u8 - Standard HLS stream endpoint for playback.

App Features

  • On-the-fly HLS Conversion - Upload standard MP4, MKV, or MOV files to automatically transcode into adaptive HLS streams.
  • Embedded Players - Easily embed video streams on external sites using an iframe element.
  • Popup Player Modal - Seamlessly preview content straight out of the system library.
  • Categorization - Tag videos with Library and Collection markers for logical grouping.
  • Upload Monitoring - Built-in progress tracking during chunked byte-transfers.
